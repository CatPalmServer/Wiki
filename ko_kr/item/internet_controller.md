## 사용
같은 이름의 트리거는 같은 이름의 감시기를 활성화하며, 같은 이름의 트리거와 감시기는 무제한으로 배치 가능합니다. 
어떤 트리거가 레드스톤 신호를 받으면 모든 같은 이름의 감시기가 활성화되어 블록이 업데이트 됩니다. ([이 API 엔드포인트](https://catpalm.gitbook.io/webapi/controller)를 사용하여 활성화할 수도 있습니다.)
